# SCR101 ホーム（TOP）画面設計書

## 1. 画面概要

### 1.1 画面の目的

* サービスの概要を提示し、ユーザーに「どんな講座があるのか」「どうやって利用するのか」を短時間で理解させる。
* 受講者を「講座詳細」「会員登録／ログイン」「無料コンテンツ」などの主要導線へ誘導する。

### 1.2 利用者ロール

| ロールID | ロール名  | 権限概要            |
| ----- | ----- | --------------- |
| R001  | 管理者   | 画面閲覧可（主に動作確認）   |
| R002  | 受講者   | 画面閲覧可           |
| R003  | 未ログイン | 画面閲覧可（講座詳細までは可） |

### 1.3 関連機能・関連API

| 種別  | ID     | 名称                    | 用途                |
| --- | ------ | --------------------- | ----------------- |
| 機能  | F001   | ホーム表示                 | TOP画面の表示          |
| 機能  | F002   | おすすめ講座一覧表示            | おすすめ講座カードの表示      |
| 機能  | F003   | 検索ボックスから講座検索          | キーワード検索で講座一覧へ遷移   |
| API | API101 | get_top_contents      | メインビジュアル文言・各種設定取得 |
| API | API102 | list_featured_courses | おすすめ講座一覧取得        |

---

## 2. 画面レイアウト設計（UI仕様）

### 2.1 ワイヤーフレーム（構造説明）

* 最上部：共通ヘッダー（LAYOUT002 共通ヘッダー／フッター設計を参照）

* 上部：共通ヘッダー

  * ロゴ
  * ナビゲーション（「講座一覧」「サービス紹介（任意）」）
  * 「ログイン」「会員登録」ボタン
* 中央上部：メインビジュアル（ヒーロー）

  * キャッチコピー
  * サービス説明文（1〜2行）
  * 「講座を探す」ボタン
* 中央：おすすめ講座一覧（カードレイアウト・横3件程度）

  * 講座サムネイル画像
  * 講座タイトル
  * 価格
  * レベル／カテゴリ
  * 「詳細を見る」ボタン
* 中央下部：サービス利用の流れ（3ステップ）

  * STEP1：講座を選ぶ
  * STEP2：購入する
  * STEP3：動画を視聴する
* 下部：フッター

  * 利用規約／プライバシーポリシーへのリンク
  * 事業者情報

* 最下部：共通フッター（LAYOUT001 共通ヘッダー／フッター設計を参照）

### 2.2 UIコンポーネント一覧

※ 共通ヘッダー／フッターのUIコンポーネント定義は「LAYOUT001 共通ヘッダー／フッター設計」を参照し、本画面には記載しない。

| UI ID | コンポーネント種類        | ラベル            | 説明                   |
| ----- | ---------------- | -------------- | -------------------- |
| UI01  | Image            | ロゴ             | サイト共通ロゴ              |
| UI02  | Button           | ログイン           | ログイン画面へ遷移            |
| UI03  | Button           | 会員登録           | 会員登録画面へ遷移            |
| UI04  | Text             | キャッチコピー        | サービスコンセプト文言          |
| UI05  | Text             | サービス説明         | 1〜2行のリード文            |
| UI06  | Button           | 講座を探す          | 講座一覧 or おすすめ一覧へスクロール |
| UI07  | Card (repeat)    | おすすめ講座カード      | 講座情報の一覧表示            |
| UI08  | Button           | 詳細を見る          | 講座詳細画面へ遷移            |
| UI09  | Text (3 columns) | 利用の流れ（STEP1〜3） | 利用フローの図解             |
| UI10  | Link             | 利用規約           | 規約ページ（別途）へのリンク       |
